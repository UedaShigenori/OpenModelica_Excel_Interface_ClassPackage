
 # OpenModelicaとExcelVBAのインターフェースクラスライブラリ

## 本プログラムについて  
ExcelVBAからOpenModelica Shellを動かすクラスライブラリです  
本プログラムでは以下の事が可能です  
- moファイルの実行  
- xmlファイルの初期値,パラメータ値変更  
- パラメータスタディ機能(1変数)  
- 

# 使用方法
Excel-OpenModelica-Interface.xlsm内に記述しています
